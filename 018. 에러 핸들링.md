## 에러 핸들링

#### error 타입  
내장되어 있는 error 타입을 이용하여 에러를 처리할 수 있습니다.  
```go
func divide(n1, n2 float64) (float64, error) {
    if n2 == 0 {
        return 0, errors.New("0으로 나눌 수 없습니다.")
    }
    return n1 / n2, nil
}

func main() {
    result, error := divide(10, 0)

    if error == nil {
        fmt.Println("10 / 0 =", result)
    } else {
        fmt.Println(error)
    }
}

// 0으로 나눌 수 없습니다.
```
